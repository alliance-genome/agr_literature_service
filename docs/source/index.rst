Alliance Literature Service 
===========================

The Alliance Literature Service is structured into two Python apps and one Javascript React web interface.
Below you can find documentation on how to start the containers required for the using, testing and developing the
:doc:`/backend_api` and more information on how to run the :doc:`pubmed_pipeline` in order to ingest the publications
served by this service.
